## COLOR-PICKER

Click the color palette and choose the color and generate your own color according to you choice. This  project is made from *React* and *CSS*.

-------------------------------

## Feature 🚀

- Click color palette
- Then choose any color
- Then move the circle with your mouse on the color
- then generate your own color according to your choice
- Copy HEX code 

--------------------------

## Technologies used 

- **React**
- **CSS**
